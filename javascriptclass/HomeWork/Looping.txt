Looping

1. Write a program to print numbers from 1 to 10 using a for loop.  
2. Create a program to print even numbers between 1 and 20 using a while loop.  
3. Write a function that takes a number as input and prints the multiplication table for that number using a for loop.  
4. Create a program to find the sum of all numbers between 1 and 100 using a for loop.  
5. Write a function to reverse a given array using a for loop.  
6. Write a program to find the factorial of a given number using a while loop.  
7. Create a program to check if a given number is prime or not using a for loop.  
8. Write a function to find the largest number in an array using a for loop.  
9. Create a program to print the Fibonacci sequence up to a given number using a while loop.  
10. Write a function to count the number of vowels in a given string using a for loop.
